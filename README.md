# 运势抽签小游戏：
支持求财 / 姻缘 / 事业 / 健康 / 随机 5 类抽签，设置单日单类型仅抽 1 次的规则，所有结果本地存储，仅供娱乐（代码由豆包、GitHub Copilot辅助生成，经人工测试并处理错误）。
目前仅在64位windows11系统上测试过。
# 使用语言:C++；使用raylib5.5版本的图形库
背景音乐以及AI解签尚未实现；
目前界面仅是简单的图形组合，字体也不够清晰；
总而言之，这个项目十分简陋。
